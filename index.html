Web Mapping

Web mapping refers to the use of the internet to display, analyze, and share geographical information in a visual format using maps. It encompasses various technologies and tools to create interactive and informative map experiences on web browsers.

OpenStreetMap

Open-source geodata platform
  • Provides free and editable geographic data used in various mapping applications.

Leaflet

Open-source JavaScript library
  • Helps developers create interactive maps on web pages.

MapTiler

Cloud-based mapping platform
  • Provides tools and services for creating custom maps and integrating them into web applications.
